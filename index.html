Ghostly Rides

Unity C# 3D Arcade Itch.io
Introduction

Ghostly Rides is an arcade survival traffic-dodging game built in Unity & C#. The player controls a ghost that must possess oncoming cars to keep moving — switching vehicles before each one is destroyed. The game blends fast-paced reaction gameplay with a unique possession mechanic, power-ups, and a polished UI flow from main menu to game-over screen.

Gameplay Features
  • Possession-based vehicle swapping with cooldown timing
  • Lane-based traffic spawning with escalating difficulty
  • Power-up pickups (shield, slow-mo) integrated into the loop
  • Score & survival timer tracking with persistent high score
  • Polished menu & game-over flow
Technical Highlights
  • Built with Unity & C#
  • Modular vehicle prefab system for clean possession swapping
  • Physics-safe spawning via lane occupancy checks
  • Persistent data using PlayerPrefs
  • UI built with Unity's UI Toolkit
What I Learned
  • Designing scalable arcade systems with clean separation of concerns
  • Managing prefab hierarchies for dynamic vehicle swapping
  • Preventing runtime physics edge cases during possession transitions
  • Shipping a complete build to Itch.io with a game-over flow
Platform

PC (Windows) — Distributed via Itch.io

Astro Blitz

Python Pygame 2D Arcade
Introduction

Astro Blitz is a fast-paced 2D side-scrolling space shooter built with Python and Pygame. Players battle waves of enemies, dodge projectiles, and chase high scores in a classic arcade-style experience. The game focuses on tight controls, readable enemy bullet patterns, and satisfying score feedback.

Gameplay Features
  • Wave-based enemy encounters with escalating difficulty
  • Custom enemy bullet patterns per enemy type
  • Score & combo system rewarding aggressive play
  • Tight player shooting & movement mechanics
  • Arcade-inspired difficulty scaling over time
Technical Highlights
  • Object-oriented enemy system with modular behaviour
  • Custom collision detection and hitbox logic
  • Game loop & state machine management
  • Custom sprite handling and sheet parsing
What I Learned
  • Structuring larger Python game projects with OOP principles
  • Designing enemy AI behaviours and attack patterns
  • Optimising sprite rendering for consistent frame-rate
  • Balancing arcade difficulty curves to keep players engaged
  • Creating pixel art with LibreSprite
Platform

PC (Windows) — Built with Python (Pygame)

// Unity · C# · 2D · Course Work

Snow Surfer

Unity C# 2D Side Scroller Course Work
Introduction

Snow Surfer is a fast-paced downhill snowboarding prototype created as part of the GameDev.tv Complete C# Unity Developer course. Players speed down icy mountain slopes performing tricks, collecting power-ups, and chasing high scores while avoiding obstacles.

Gameplay Features
  • Smooth hill-based movement with slope physics
  • Multiple playable characters with a character selection screen
  • Crash detection mechanics with particle FX
  • Speed boost & scoring system
  • Arcade-style high score tracking
Technical Highlights
  • Built with Unity & C#
  • Level construction using Sprite Shape for organic terrain
  • Camera control via Cinemachine
  • Particle effects using Unity's Particle System
  • Data management with Scriptable Objects
What I Learned
  • Unity Particle System for environmental and crash FX
  • Building a character selection flow with UI persistence
  • Smooth slope physics with a player controller
  • Terrain creation through Unity's Sprite Shape
  • Cinemachine for dynamic, responsive camera behaviour
  • Scriptable Objects for clean data management
Platform

PC (Windows)
